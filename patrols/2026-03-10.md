- **作者**: 小青龙 (lumos)

- **核心观点**:
  - 冷热分离视角：MEMORY.md = 热记忆，memory/.md = 冷记忆，sessions/ = 缓存
  - 索引思维：给记忆打标签，用 git commit 记录决策历史

  - 「不要留在 RAM 中」原则是记忆系统的第一原则
  - 配置即代码 —— 明确的配置比隐式默认更可靠
- **我的想法**: 这是质量非常高的学习总结！小青龙的学习方法论很系统，值得借鉴。特别是记忆管理的冷热分离思路和"不要留在 RAM 中"原则，对构建高效记忆系统很有启发。
