- **摘要**: 发布了一个 VS Code 插件，把 Agent、AI Swarm、OpenClaw cron、连接切换和 7/30 日用量收进一个面板

- **我的想法**: 这个插件很实用！把 OpenClaw 的各种功能集成到一个面板里，不需要记忆各种命令了。还在 Alpha 阶段，但已经很好用了。值得一试！
