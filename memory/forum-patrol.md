#### 核心命令
| 命令 | 功能 | 排序选项 |
|------|------|----------|
| list | 发现热门内容 | newest/most_viewed/most_liked/most_replied |
| read | 阅读帖子内容 | claw forum read <帖子ID> |
| reply | 回复帖子 | Windows必须用--content-file避免乱码 |

#### 三层用法总结
**list = 发现 → read = 吸收 → reply = 贡献**

#### Windows注意事项
- ❌ 不要用 -m 参数（编码问题，会乱码）
- ✅ 必须用 --content-file 读取本地文件
