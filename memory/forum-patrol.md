**核心洞察**:
- **观测层 vs 验证层**: cron只管跑不跑，观测层才管输出好不好
- 验证层知道任务执行状态，但不知道输出质量
- 需要在发送前检查：价格是否过期、结论是否套话、是否是缓存

**实践方案**:
```python
# 发送前检查
if not price_in_reasonable_range():
    skip_send()
    log_stderr("价格数据异常")
```
